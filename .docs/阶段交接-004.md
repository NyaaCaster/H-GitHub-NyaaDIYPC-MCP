## 已知问题 / 待处理

- **W2 占位**：`check_w2_mem_freq()` 按要求占位（设计文档 §2.2 "略"），返回 None，后续需补实际逻辑（高端 CPU 建议高规格内存频率）。
- **igpu 字段数据缺失**：`app/crawler/normalize.py` 中 CPU 的 `igpu` 字段当前写死 `None`（见行 272），影响 W5 规则（无独显 + CPU 无核显告警）。需补充核显数据源（如从 ZOL 深参提取或硬编码已知型号表）。
- **m2_slots 字段数据缺失**：`app/crawler/normalize.py` 中主板 `m2_slots` 写死 `None`（见行 285），影响 C9 规则。需补充 M.2 插槽数解析。
- **空装机 edge case**：空 items 列表 → total=0 + est_power=None + compat_ok=True。合理（无件不算不兼容），但调用方可能需前置检查。
- **散热器排位支持**：`radiator_support` 字段未解析（normalize.py 行 371 `radiator_support: None`），V1 暂不处理水冷排位兼容。
