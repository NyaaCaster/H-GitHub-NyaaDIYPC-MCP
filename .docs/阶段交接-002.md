## 已知问题 / 待处理

- **未做真机抓取验证**：P2 DoD 要求对 CPU + 显卡做小规模干跑，但当前 macmini 未部署、Windows 未启动容器（铁律：禁止未经要求启动）。建议 P7 部署后在 macmini 上跑 `docker compose exec app python -m app.crawler.run --category cpu` 验证端到端落库。
- `normalize.py` 的归一表（socket/板型/认证等）基于常见值构建，实际抓取后可能遇到新变体，届时迭代补充。
- `parampage.py` 深参批量抓取是**串行**（逐个 await），如需加速可改为 `asyncio.gather` + semaphore 并发，但当前限速策略下串行已够用。
- `tianti.py` 的 GPU HTML 解析依赖 `.item-box` class 结构和 `.num` class，若 ZOL 改版需更新正则。
- `mark_inactive` 在单个品类 0 条目时跳过（保护已有数据），但全品类为空时不会标停售。
