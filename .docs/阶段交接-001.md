## 已知问题 / 待处理

- P1 使用 `FastMCP` 而非新版 `MCPServer`（SDK 1.28.1 兼容）。API 参数（`version`/`json_response`）在 `__init__` 和 `streamable_http_app()` 中不可用，需 `run()` 传递。当前不影响功能。
- auth.py 的 `BearerAuthWrapper` 不再提供 ASGI wrap（鉴权逻辑内联到 `server.py`）。后续如需复用可提取。
- schema.py 未在容器启动时自动调用 `init_db`——需在 P2 爬虫首次运行时调用，或添加 startup hook。
