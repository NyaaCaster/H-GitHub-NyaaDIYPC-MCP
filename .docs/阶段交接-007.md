## 运维速查

```bash
# macmini 上手动爬虫
docker exec nyaadiypc-mcp python -m app.crawler.run --category all

# 查看爬虫日志
docker exec nyaadiypc-mcp tail -100 /app/data/crawler3.log

# 重启服务
cd /root/DockerContainer/NyaaDIYPC-MCP && python3 restart.py

# 查看 crontab
crontab -l | grep nyaadiypc

# 验证 MCP 鉴权
curl http://localhost:5115/health
curl -X POST http://localhost:5115/mcp -H 'Authorization: Bearer <token>' ...
```
